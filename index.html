Assumption

Why assumption is important? Becasue all algorithms are based on the assumption. You must make sure the assumption matches the pratical problem when cloosing the model
Algorithm Assumption Type
KNN Similar inputs have similar outputs Classification or Regreesion ---- Both
Perceptron Data is linearly separable Binary classification (y is in {1,-1}) ---- Classification
Naive Bayes Given the label, feature values are independent Naive Bayes is a linear classifier (generative learning P(x|y)) ---- Classification
Logistic Regression Model P(y|x) directly. We don't care about the model of P(x|y) any more because we estimate w,b with MLE or MAP to maximize P(y|x) Classification (discriminative P(y|x) counterpart of Naive Bayes )
SVM Linear separately + Binary classification Linear classifier
Linear Regression Data is drawn from line wTx. For each data point with feature xi, the label is drawn from Gaussian distribution Regression
Gaussian Process Regression Data is all Gaussian distribution (Put the kernel matrix into Gaussian distribution) Regression gives both prediction and certainty
